# Adhoc Internship - Login & Signup Project

A clean, modern, and secure user authentication web application featuring fully functional Login and Signup pages built with React.

---

## 🚀 Features

* **User Registration:** Secure signup form with real-time input validation.
* **User Authentication:** Secure login functionality with state management.
* **Component-Based Architecture:** Built using reusable React components.
* **Error Handling:** Clear, user-friendly error messages for invalid inputs or incorrect credentials.

---

## 🛠️ Tech Stack

* **Frontend:** React.js, JSX
* **State Management:** React Hooks (`useState`, `useEffect`)
* **Routing:** React Router DOM
